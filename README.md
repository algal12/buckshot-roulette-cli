# Buckshot Roulette (CLI)

A small Python terminal game where you face a dealer in a revolver-style survival challenge.  
There is **one bullet** hidden in a **6-chamber revolver**. Each turn you choose whether to shoot **yourself** or **the dealer**. Survive **3 rounds** to win.

> Note: This is a simple learning project focused on loops, randomness, and game-state logic.

## Features
- Random bullet position (1 bullet, 6 chambers)
- Choice-based gameplay (shoot yourself or the dealer)
- Dealer turn logic (dealer may shoot you depending on your choice)
- Win condition: survive 3 rounds
- Lose condition: bullet fires on your turn or the dealerâ€™s turn
